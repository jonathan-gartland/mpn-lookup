## Try it out:

```bash
1. git clone https://github.com/jonathan-gartland/mpn-lookup.git
2. cd mpn-lookup
3. npm i
4. ts-node
5. > import { getQtMpn } from './mpn.lookup'
6. > getQtMpn(0)
```
